# Account App

## Capabilities
- Add Todo
- Get Todos
- Sort Todo list
- Edit Todo (admin only)
- Sign in as admin
- Sign out
- Pagination

## Worth to mention

### .env file must contain the following variables:
- DATABASE_URL
- APP_PORT

## Technology stack
- React
- Redux, Redux Toolkit
- NodeJs
- Experess
- PostgreSQL
- Other libraries
